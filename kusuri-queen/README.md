# クスリクイーン 👑💊

看護師の実務目線でつくった医薬品リファレンスアプリ。

薬の名前(一般名・商品名)、副作用、相互作用(飲み合わせ)、点滴の配合変化、
塗り薬・点眼薬の使い方、使用上の注意、看護のポイントを、現場で素早く引けます。

## 特徴

- 🔍 **即時検索** — 一般名・商品名・ひらがな・分類名(例:「ろきそ」「ラシックス」「PPI」)で検索
- 🗂 **カテゴリ別** — 内服 / 注射・点滴 / 外用 / 点眼
- 🔀 **相互作用チェッカー** — 2剤を選んで収録データの相互作用・配合変化を照合
- 💧 **IVルール** — 配合変化、希釈濃度、投与速度(例:KCLは40mEq/L以下・20mEq/h以下)
- ⚠️ **ハイリスク薬バッジ** — インスリン、KCL、抗凝固薬などを明示
- ★ **お気に入り** — よく使う薬を端末に保存(localStorage)
- 📱 **モバイル最適化 + PWA** — ボトムタブ・セーフエリア対応、ホーム画面に追加可能

## 技術スタック

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- 依存最小・完全クライアントサイド(サーバー不要)

## 開発

```bash
cd kusuri-queen
npm install
npm run dev      # 開発サーバー
npm run build    # 型チェック + 本番ビルド (dist/)
npm run preview  # ビルド結果のプレビュー
```

## App Store 配信への道筋

本アプリはPWA対応済みのWebアプリです。App Storeで販売する場合は
[Capacitor](https://capacitorjs.com/) で `dist/` をラップしてiOSプロジェクト化するのが最短です:

```bash
npm i @capacitor/core @capacitor/ios
npx cap init "クスリクイーン" com.example.kusuriqueen --web-dir dist
npx cap add ios && npx cap open ios
```

> 医療情報を扱うアプリはApp Storeの審査(ガイドライン1.4 身体的危害)で
> 情報源の明示・免責表示が求められます。公開前に収録データの監修体制を整えてください。
